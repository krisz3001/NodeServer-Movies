Version: 2.0
Default port: 3001
Release date: 2022.02.13.

How to use: 
1. Run run.bat to set up the program.
2. Place video files in the 'media' folder in .mp4 format (with audio in .mp3).
	In case of series, name the file like this: seriesname_sX_eY.mp4
	Where X = number of the season in the series and Y = number of the episode in the season
3. Type localhost:3001 in the browser.
4. Add elements like below:
Movies:
	Teljes név: <anything>
	Fájlnév: <name of the file in the 'media' folder without extension>
Series:
	Teljes név: <anything>
	Fájlnév: <name of the file in the 'media' folder without season, episode information and extension>
	Évadok száma: <number of seasons in the series>
	Epizódok száma: <number of episodes in a season>
